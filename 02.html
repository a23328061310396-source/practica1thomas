Tipos de Malware

Virus: Programas que se insertan en archivos legítimos para propagarse.

Gusanos: Se replican automáticamente y se propagan en redes.

Troyanos: Se disfrazan como programas legítimos, pero dañan o roban información.

Spyware: Espía la actividad del usuario y roba datos.

Ransomware: Bloquea el acceso a los datos y pide un rescate.

Principales Ciberataques

Phishing: Engaña al usuario con correos o sitios falsos.

DDoS: Sobrecarga servidores para interrumpir servicios.

Ingeniería Social: Manipula a las personas para obtener información.

SQL Injection: Inserta código malicioso en consultas a bases de datos.

Man-in-the-Middle: Intercepta la comunicación entre dos partes.
